TimeSheet App

A simple TimeSheets web app to add employee clock in/out information for manager approval

Technologies Used -->
  • Front End: HTML, CSS, JavaScript, Bootstrap 4
  • Back End: Python, Django
  • Database: PostgreSQL
Description ---------->
  • Currently, there is one admin user and two regular users
  • New users can register and add their clock in/out information
  • Custom Admin page to manage the application
  • Only admin user can approve the information and view all user information
  • Each user has their own dashboard displaying their clock in/out information along with their salary and approval status
